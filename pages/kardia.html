Hello and welcome! This is the preview of Kardia, a simple about page I release under @pneuma-themes and coincidentally, my first release! Yay!

This is a very simple about page that has boxes where you can put literally anything. I put a personal information, updates and a custom link box where I put custom links. You can, of course, customize it to your liking. I have put notes inside the code to edit, so please make sure you have thoroughly read the code! I hope my notes are understandable but if not, do feel free to send me an ask.

Kardia (κᾰρδῐ́ᾱ, (kardíā)) is an Ancient Greek term referring to heart as the source of feeling and emotions, and used in modern age as a term referring to the medical term of heart.

Features included in this page are as the following:

  • The ability to add as many boxes as you want.
  • The ability to write your about as long or as short as you want.
  • A quote above the about text.
  • Customizable icons, please refer to the notes I included in the code on how to change it!
  • Sidebar includes a custom icon, a title and a description where you can write as long or as short as you want.
  • Customizable colors for light and dark mode.
As this is a page theme, posts on your blog will not be displayed.

I think that about covers it. Thank you so much if you choose to use this page, and I hope you enjoy it! If you have any questions, do leave me an ask, and if you like this page, please like and reblog the theme post!
